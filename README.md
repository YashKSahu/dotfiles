# Dotfiles

My personal macOS configuration files for building a keyboard-driven, tiled desktop environment.

## Repo Contains

* **`.config/aerospace/`**: Configuration for window layouts, sizing constraints, workspaces, and navigation hotkeys.
* **`.config/sketchybar/`**: Shell-scripted status bar widgets tracking system vitals (CPU, RAM, GPU, Disk, Battery, Volume) and active spaces.
